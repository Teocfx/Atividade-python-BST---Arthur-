# 🌳 Binary Search Tree (BST) em Python

Implementação de uma Árvore Binária de Busca em Python utilizando dicionários, contendo as operações propostas na atividade.

## 📚 Funcionalidades

- Inserção de nós
- Busca de elementos
- Remoção de nós
- Percurso In-Order
- Cálculo de altura da árvore

## ⚙️ Tecnologias

- Python 3
